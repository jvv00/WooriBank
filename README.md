## :desktop_computer: 프로젝트 개요
기존 웹사이트의 경우   
좁은 헤드 크기에 비해 과밀되어 있는 메뉴, 직관적이지 않은 아이콘, 세부 메뉴 별 통일되지 않은 UI 등으로 인해 가독성이 떨어진다는 단점이 있습니다.   
이에 아쉬움을 느껴 다양한 연령층이 쉽고 편리하게 사용할 수 있도록 리뉴얼 디자인 하는 프로젝트를 진행 중입니다.
